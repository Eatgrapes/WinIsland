# 什么是 WinIsland？

WinIsland 是一个为 Windows 桌面打造的灵动岛实现。受 iOS 启发，它提供了一个集中的枢纽，用于媒体控制以及各种工具，同时拥有流畅且不唐突的设计。

## 核心特性

- **媒体控制**：查看专辑封面、歌曲信息以及播放状态。
- **小组件**: 自定义在展开界面显示的小组件，可以显示各种信息。
- **通知显示**: 可以把Windows提醒的通知显示在WinIsland上。
- **轻量级**：使用 Rust 编写，占用极低的系统资源。

## 为什么选择 WinIsland？

与其他“灵动岛”克隆产品不同，WinIsland 专注于成为一个实用工具，而不仅仅是一个视觉噱头。
