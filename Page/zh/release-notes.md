# 发布说明

WinIsland 各版本的重要变更记录。

## 1.0.0 - 初始版本

新增：

- Windows 上的 Dynamic Island — 复刻 iPhone Dynamic Island 体验
- System Media Transport Controls (SMTC) 集成，显示正在播放的媒体
- 实时歌词显示，与媒体播放同步
- 音频频谱可视化
- 弹簧物理动画，岛体展开/收起流畅自然
- 主题配色系统，支持自定义颜色
- 自定义字体支持
- 玻璃效果（亚克力/模糊背景）
- 多语言 i18n 支持（英语、中文）
- 设置界面，支持热重载配置
- 插件系统基础（C ABI、PluginVTable、内容渲染）
- ContextManager 统一内容调度器
- Nightly 自动更新系统
- **插件媒体源**（`MediaSourceC`）— 插件可注入完整的媒体数据（标题、艺人、专辑、进度、封面）
- **GitHub Releases API** 集成，更新下载更可靠
- **编译时构建时间戳**嵌入二进制 — 手动替换 exe 后不再反复提示更新
- **HTTP 重试机制**（3 次，指数退避）用于更新下载
- **官网发布说明页面**

修复：

- "无法下载新版本"错误现在会自动重试后再显示失败对话框
- 手动替换 exe 不再导致反复提示更新
- 更新下载对网络故障更具弹性
