# Download WinIsland

You can download either the stable official releases or the nightly builds below.

## Official Releases

Stable builds that are suitable for most users.

[Download Latest Release (.exe)](https://github.com/Eatgrapes/WinIsland/releases/latest/download/WinIsland.exe)

## Nightly Builds

Nightly builds are automatically generated every time we update the code. They include the latest features but might be less stable than official releases.

[Download Latest Nightly (.exe)](https://github.com/Eatgrapes/WinIsland/releases/download/nightly/WinIsland.exe)

---

### Installation

1. Download the `WinIsland.exe`.
2. Move it to a folder where you want to keep it.
3. Run the executable.
4. (Optional) Add a shortcut to your `Startup` folder to launch it automatically.

### System Requirements

- Windows 10 or 11
- 64-bit architecture
- Modern graphics card with Skia support (most are supported)
