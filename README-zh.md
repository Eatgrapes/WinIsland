# WinIsland

[English](./README.md) | 简体中文

> [!WARNING]
> 该项目仍在开发中 (WIP)，可能会出现错误。

这是一个可以在 Windows 上显示**灵动岛**的项目。

> **基于 [Eatgrapes/WinIsland](https://github.com/Eatgrapes/WinIsland) 的优化版本**，包含多项修复和改进。

## 下载

### 优化版本（推荐）
从 [Release](https://github.com/RSLN-creator/WinIsland_Small-optimization/releases) 下载

### 原版
从 [原版 Release](https://github.com/Eatgrapes/WinIsland/releases) 或 [官方网站](https://tanikaze.icu/WinIsland) 下载

## 更新内容

- 修复 SMTC 封面切歌后不更新问题
- 修复动态颜色与封面同步问题
- 新增设置窗口单实例检测
- 新增字体对比显示功能
- 修复设置窗口居中显示（DPI 缩放适配）
- 设置界面主题跟随系统

## 构建项目

- **Rust** 环境
- **Cargo**

```cmd
git clone https://github.com/RSLN-creator/WinIsland_Small-optimization.git

cd WinIsland_Small-optimization

cargo build --release
```
