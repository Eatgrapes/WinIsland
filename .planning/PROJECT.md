# WinIsland

## What This Is

WinIsland 是一个 Windows 桌面灵动岛应用，在屏幕顶部以动画胶囊形式实时展示系统媒体播放信息（封面、标题、频谱、歌词），支持交互控制。纯 Rust + Skia 自绘 UI，灵感来源于 iPhone 灵动岛设计。

## Core Value

**用户无需切换窗口即可在桌面顶部看到当前播放歌曲的封面、歌词，并进行播放控制。**
